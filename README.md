## Key Features

### AI-Powered Agriculture Solutions
- **Disease Detection:** Integrated **2 pre-trained AI models** to analyze crop images and detect diseases early.
- **Smart Recommendations:** Provides intelligent fertilizer and irrigation advice based on real-time analysis of soil data and **Weather API** integration.
- **Yield Optimization:** Data-driven insights to maximize crop output.

### E-Commerce & Marketplace
- **Crop Listings:** A complete marketplace supporting **50+ crop categories** with advanced search and filtering capabilities.
- **Plant Gifting:** A dedicated section for gifting plants and agricultural products.
- **Responsive UI:** Modern, user-friendly interface designed for optimal experience across desktop and mobile devices.

### Community & Communication
- **Real-Time Chat:** Integrated buyer-seller communication system for seamless negotiations.
- **Q&A Forum:** An expert community forum with threaded discussions to solve farming queries.

## Tech Stack

This project uses the **MERN** stack architecture involving **12+ API endpoints** and **20+ reusable React components**.

- **Frontend:** React.js, HTML5, CSS3, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **AI/ML Integration:** Pre-trained Machine Learning Models
- **External APIs:** OpenWeatherMap API (or specific Weather API used)
